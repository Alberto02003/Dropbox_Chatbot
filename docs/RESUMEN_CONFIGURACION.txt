  PASOS PARA USAR LA APLICACIÓN
═══════════════════════════════════════════════════════════════════

PASO 1: Configurar Hosts File (Una sola vez)
---------------------------------------------
Windows: Editar C:\Windows\System32\drivers\etc\hosts como Admin
Linux/Mac: sudo nano /etc/hosts

Añadir línea:
127.0.0.1 dropboxaiorganizer.com

PASO 2: Configurar Variables de Entorno
----------------------------------------
Crear backend/.env con:
DROPBOX_APP_KEY=tu_app_key
DROPBOX_APP_SECRET=tu_app_secret
GEMINI_API_KEY=tu_gemini_key
FRONTEND_URL=https://dropboxaiorganizer.com

PASO 3: Iniciar Aplicación
---------------------------
Windows:
  - Desarrollo: start-dev.bat
  - Producción: start-prod.bat (como Administrador)

Linux/Mac:
  - Desarrollo: ./start-dev.sh
  - Producción: sudo ./start-prod.sh

PASO 4: Acceder
---------------
Desarrollo:  https://dropboxaiorganizer.com:5173
Producción:  https://dropboxaiorganizer.com  ← SIN PUERTO!
